    This website focuses to provide business platform for schools with low-budget spending needed. Through this approach we create interactive and dynamic web application that able to function onto cloud computing which uses most effective budget output where it provides various benefits. During the research we came out with result of Microservice Architecture, Use Case Diagram, System Flowchart, Cloud Architecture, performing analysis of website using X-Ray Service and Elastic Beanstalk.

    Project creation involved wtihin Visual Studio Code software that uses combination of ASP.Net, AWS Cloud service such as RDS, S3, DynamoDb and X-Ray while within the application we utilized the use of PaaS tool, Elastic Beanstalk to conduct the deployment process.

    Functionality of the system for user being able to conduct register and login onto the website that provide them different interface where for teacher they have ability of managing which teacher will teach the targeted subject, editing the data, create data for subject that student will choose, and seeing teacher data within the website even creating payment record to be shown for them which student have conduct subject payment.While for student they can request which subject access to be purchase where they will require the approval from the teacher after conducting the request.
