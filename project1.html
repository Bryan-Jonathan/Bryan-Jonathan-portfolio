    Bakery Shop is one of my college project that was created during the pandemic situation where the requirement of people being able to do online orders need to exist within the situation. During the research gathering for the project we came out with result shown on design above such as Audience Classification, Customer Entity Data Relationship, Website Wireframe, Customer Navigational Structure, and result of Bakery Website.

    The project creation involved within Visual Studio Code software with combination of C#, HTML and CSS coding. Functionality from the website consist from bakery delights advertising, location of store, feedback from customer and login system for user and admin with their ability of either ordering and manage the website.
