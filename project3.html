    Android File System is project design personally with the goals of creating application that able to encrypt and decrypt data making the data more protected with the help of Base64 encryption or decryption. The other protection was implemented onto the login and register system where the password have protection of hash and salting that will automatically transform password from being readable into unreadable data and differentiating data that have same result.

    Project creation involved within Android Studio software that uses help of PHP coding, XAMPP, Apache and MySQL to run overall application system and providing database system. Functionality of the application able to provide register and login system for user that give them ability of encrypt or decrypting data they desire to be protected from being accessed by attacker.
